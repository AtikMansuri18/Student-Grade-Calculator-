# 🎓 Student Grade Calculator (Java)

## 📌 Description
This is a Java-based application that calculates student grades based on subject marks.

## 🚀 Features
- Input student name and subject marks
- Calculates total and average
- Assigns grades (A–F)
- Uses Arrays and Classes
- Uses StringBuilder for formatted output
- File handling to store data
- Exception handling for invalid input

## 🛠️ Technologies Used
- Java
- OOP Concepts
- File Handling
- Exception Handling

## ▶️ How to Run

Compile:
javac com/Student.java calculate/gradecalculater.java

Run:
java com.Student

## 📂 Output
- Displays result on console
- Saves result in `a1.txt`
